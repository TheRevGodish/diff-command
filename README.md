Algorithms
==========

The following project is focus on comparing two files, two directories or two strings,
emulating the real _**diff**_ command.

# **OPTIONS**:

- ##### TYPE ALGO:
  - **-n**   --> 2D matrix
  - **rien** --> Myers
- ##### TYPE ENTRÉE:
  - **-s**   --> 2 strings
  - **-r**   --> recursive on 2 directories
  - **rien** --> 2 files
